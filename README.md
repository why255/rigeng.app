## 项目一句话

> **日耕朝夕，耕愈工作，耕暖生活**
>
> 日耕 APP 是用户的土地，日耕驾驶舱是用户的大脑。
> 在这片土地上，帮用户春播、夏种、秋收、冬藏，放大专业价值。

---

## 6 大驾驶舱（按业务价值链）

```
采撷（采集内容）→ 智造（创作内容）→ 分发（发布内容）
                → 获客（引流量）→ 私域（激活）→ 转化（成交）
```

| # | 驾驶舱 | 品牌语 | 一句话 |
|---|---|---|---|
| 1 | **内容采撷舱** | 内外双通道，好料自然来 | 私有库 + 外部爆款双通道 |
| 2 | **内容智造舱** | 素材随手耕，精品自然出 | 一键生成多形态精品内容 |
| 3 | **全网分发舱** | 规则配一次，全网自动耕 | 多平台半自动发布 |
| 4 | **精准获客舱** | 钩子轻轻放，用户自然来 | 全自动加好友 + 活码归因 |
| 5 | **私域耕耘舱** | 存量轻轻耕，激活有温度 | 拟人化群发 + 朋友圈智能互动 |
| 6 | **成交转化舱** | 信任先播种，深耕扎下根 | 客户分层 + 训练营 + 跳转日耕 APP |
