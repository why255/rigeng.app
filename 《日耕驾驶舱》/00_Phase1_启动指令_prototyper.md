| 情况 | 应对 |

| prototyper 工具不是 MiniMax Code | 立即停止，回弹给 Mavis |
| 出现"AI / 赚钱 / 变现 / 推广"等禁用词 | 立即自审替换 |
| 任何产品决策要由 AI 拍板 | 拒绝决策，回弹给安老师 |
| 业务链顺序被提议调整 | 拒绝变更，回弹给安老师 |
| 红线模块（chat db / 自动加好友 / 朋友圈自动化）原型实现细节 | 必须王昊宇 + 安老师共同拍板 |
| UI 视觉与 ui-designer 组件库不一致 | 立即停下对齐组件库 |
