## 紧急情况

| 情况 | 应对 |
|---|---|
| backend-coder 工具不是 MiniMax Code | 立即停止，回弹给 Mavis |
| 出现 AI/赚钱/变现/推广等禁用词 | 立即自审替换 |
| 任何产品决策要由 AI 拍板 | 拒绝决策，回弹给安老师 |
| 业务链顺序被提议调整 | 拒绝变更，回弹给安老师 |
| 红线模块原型实现细节 | 必须王昊宇 + 安老师共同拍板 |
| 数字人质量不达标 | Phase 3（内容智造舱）评估切云端 API |
| 微信协议升级导致失效 | 抽象一层"微信适配器"，协议变化只改适配器 |
| 抖音/小红书封号 | 风控引擎必须到位；提供"人工接管"模式 |
