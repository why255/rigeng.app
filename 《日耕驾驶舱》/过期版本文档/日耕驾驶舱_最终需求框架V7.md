### 模块一：品牌定位舱

**品牌语：履历细细耕，IP自然成**

**耕愈工作**：搞清楚"我在HR圈是谁"，让专业被看见，建立职场辨识度。
**耕暖生活**：个人品牌是变现的根基，品牌立住了，价值才能自然来。

#### 用户视角

"我作为日耕的用户，第一次进来，系统要帮我一次性搞清楚：我在HR圈子里是谁？我做过什么？我擅长什么？做成过什么？与其他HR经理人最大的不同是什么？不要让我填表，要像跟小耕聊天一样自然交流。聊完之后，系统要给我一个丰满立体的个人品牌定位。"

#### 要做的动作（用户侧）

1. **进入对话式IP访谈**：像跟小耕聊天一样，自然回答系统提出的问题
2. **接受智能追问**：系统根据我的回答智能追问，挖掘更深层的个人品牌素材
3. **查看IP定位报告**：查看系统生成的个人IP画像（人设标签、专业领域、差异化卖点、视觉风格）
4. **确认并生成IP资产**：一键生成IP名片、个人简介模板、专业背书话术、成长故事

#### 核心功能

通过后台可配置的多轮对话引擎，智能系统像专业品牌顾问一样与用户自然交流，根据用户回答实时分析并追问，最终输出丰满立体的个人品牌定位方案，为后续所有内容传播奠定基调。

#### 功能支撑清单

| 功能点       | 说明                                            |
| --------- | --------------------------------------------- |
| 对话式信息采集引擎 | 多轮自然语言交流，完全替代填表                               |
| 后台话术配置中心  | 管理员可在后台配置对话话术、追问逻辑、问题顺序，灵活调整                  |
| 智能实时分析引擎  | 根据用户回答内容实时分析，判断信息丰满度，决定是否需要追问                 |
| 智能追问生成器   | 基于IP打造目标，自动生成追问问题                             |
| 履历结构化解析   | 自动提取关键经历、成就事件、专业标签、个人故事线                      |
| IP定位算法    | 识别专业领域+差异化卖点+人设风格（温润型/专家型/陪伴型/转型型等）           |
| IP丰满度评估   | 评估IP形象是否立体丰满，不足时自动引导补充                        |
| IP名片生成器   | 自动生成图文版个人简介（适配公众号/视频号/小红书）                    |

### 模块二：内容采撷舱

**品牌语：内外双通道，好料自然来**

**耕愈工作**：从日耕私有知识库+外部平台采集高价值内容，持续学习成长，为专业输出储备弹药。
**耕暖生活**：好料是精品内容的前提，内容质量高了，变现才有底气。

#### 用户视角

"我要做内容，但不知道写什么。系统要从两个地方自动找料：一是我平日用日耕APP产生的私有知识库（我的复盘、问答、方案），二是外部平台上HR领域的爆款内容。对标博主这件事不要让我做，由后台统一配置，我只需要确认分类和筛选规则。找来的料要自动筛选、分类、转成我能用的格式。"

#### 要做的动作（用户侧）

1. **开启内部通道**：一次性授权系统调取日耕APP私有知识库（复盘记录、问答结晶、方案文档）
2. **确认外部筛选规则**：设置筛选门槛（点赞量、评论量、时间等）
3. **确认分类标签**：选择HR模块分类（招聘/培训/薪酬/绩效/员工关系/人力资源规划等）
4. **一键采集**：系统自动抓取、去重、转录、格式化为Markdown

#### 核心功能

打通日耕内部知识资产与外部公域爆款内容的双向通道。博主对标由后台统一配置，用户无感知。自动完成采集→筛选→分类→转录→格式化，让用户"无感"就能获得高质量内容素材库。

#### 功能支撑清单

| 功能点            | 说明                                  |
| -------------- | ----------------------------------- |
| 日耕私有知识库API     | 一次性授权，自动抓取用户私有知识资产（复盘、问答、方案）        |
| 后台博主对标配置中心     | 管理员在后台配置对标博主列表，用户端完全无感知，不暴露给用户      |
| 多平台爬虫引擎        | 支持抖音、视频号、B站、小红书、公众号内容采集（仅采集公开可下载内容） |
| 博主内容监控         | 自动监控已配置博主的最新内容更新                    |
| 智能筛选引擎         | 按点赞、评论、转发、收藏、时间等维度排序筛选              |

| 视频→音频提取        | 自动分离视频音轨                            |

### 模块三：内容智造舱

**品牌语：素材随手耕，精品自然出**

**耕愈工作**：将专业经验一键输出为精品内容（图文/短视频/播客），全网放大专业影响力，让老板和同行看到你的专业。
**耕暖生活**：精品内容是变现的前提，内容质量是信任的起点。

#### 用户视角

"料有了，但我没时间写、没时间拍。系统要帮我一键生成各种形态的内容：发小红书的图文、发公众号的长文、发抖音的短视频（用我的智能数字人）、发播客（用我的声音克隆），还有每天发的朋友圈文案。前几次我确认一下风格偏好，系统根据数据置信度自动判断什么时候切换全自动，后面不需要我每次审核。如果我觉得不满意，我要能提修改建议，系统要记下来，下次按我的要求改——我的要求优先于后台规则。另外，文章里怎么加微信、怎么植入我的观点，这些由后台统一配置格式范式，同时系统还能调用我私有知识库里的观点案例和安权老师的公有观点，我自己不用管。还有，现在智能时代了，内容要符合GEO（生成式引擎优化）的要求，针对ChatGPT、文心一言、DeepSeek、豆包这些主流平台做差异化优化，让智能搜索引擎能抓取推荐。"

#### 要做的动作（用户侧）

1. **前期风格确认**：选择素材→选择产出形态→选择目标平台→确认智能生成内容→确认风格偏好
2. **系统自动切换全自动**：系统根据数据置信度判断，自动切换为全自动生成模式
3. **后续查看或微调**：系统直接生成内容，用户只需查看或微调
4. **反馈与沉淀**：对不满意的内容提修改建议，系统记录为"铁律"，永久记忆，用户铁律优先于后台规则
5. **智能数字人/声音一次性授权**：首次确认声音克隆、形象克隆效果，后续自动使用
6. **暂停全自动**：用户可随时切回人工确认模式（保留开关）

#### 核心功能

基于素材自动生成多形态、多平台适配的内容资产，集成GEO差异化优化（适配ChatGPT/文心一言/DeepSeek/豆包）、开源智能数字人、声音克隆、智能配图、自动剪辑。通过系统根据置信度自动切换全自动模式 + 用户反馈沉淀机制（修改建议→永久记忆→螺旋优化，用户铁律优先），实现"越用越懂用户"的无感内容生产。广告植入（加微信、观点植入）由后台统一配置格式范式，同时智能调用用户私有观点库和公有知识库。保留"暂停全自动"开关和"紧急制动"机制。

**耕暖生活**：从免费体验到高价锁销，知识技能经验产品化、产品市场化，通过市场实现价值最大化，收入自然来，日子更温暖。

#### 用户视角

"系统要帮我自动识别高意向的人，按客户类别自动分层运营。客户分成：普通好友→免费体验用户→低价成交用户→中价追销用户→高价锁销用户（合伙人）。如果判断某用户没有挖掘价值，自动移到僵尸标签，不删除只换标签。每一类怎么进阶、匹配什么产品，后台配置好，系统自动运行。合伙人需要人工审核，但日常进阶靠积分制（我在日耕APP的使用时长、功能深度、产出、推荐用户都算积分）。另外，我可以创建训练营活动（如绩效训练营、顾问转型营），自动拉群、打卡统计、作业收集，从活动报名到会员升级的全链路追踪。我只需要在用户端看结果：一周/一个月下来，成交了多少单，每类客户多少，价值多少，提成比例多少，价值总额多少。积分规则和进阶规则我在日耕APP里也能看到。"

#### 要做的动作（用户侧）

1. **查看客户分层看板**：系统根据互动评分和行为自动分类的客户列表
2. **查看自动化运营状态**：系统在自动推送什么产品、什么内容给哪类客户
3. **查看积分与进阶**：查看自己在日耕APP的积分、进阶规则、当前层级
4. **创建/参与训练营**：创建训练营活动或报名参加，查看打卡和作业
5. **查看成交结果**：成交数、各类客户数量、每类产品成交额
6. **查看价值明细**：按产品、层级、比例计算的价值明细
7. **申请提现**：将价值收益提现到个人账户

#### 核心功能

客户自动分层（漏斗递进）→进阶规则自动匹配→产品自动推送→成交自动转化→价值自动计算。合伙人需人工审核，日常进阶靠积分机制（使用时长、功能深度、产出、推荐用户）。积分规则和进阶规则后台可配置，日耕APP和运营端均可见。用户端高度无感，只看结果。新增训练营运营支持，形成从活动报名→训练营参与→会员升级→私教转化的全链路。
